Get the color at a specific point on the surface (for procedural textures)

Parameters
pointThe point on the surface
Returns
The color at that point

By default, returns the diffuse color. Override this in derived classes to implement procedural textures like the chessboard pattern.

Reimplemented in ChessboardMaterial, and PerlinNoiseMaterial.
