\doxysubsection{Detailed Description}
A class to represent a translation transformation. 

This class is used to represent a translation transformation in the raytracer project. It inherits from the \doxylink{class_i_transformations}{ITransformations} interface. \begin{DoxyNote}{Note}
The \doxylink{class_translation}{Translation} class is responsible for applying translation transformations 
\end{DoxyNote}
