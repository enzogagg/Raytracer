\doxysubsection{Detailed Description}
A class to represent a \doxylink{class_rotation}{Rotation} transformation. 

This class is used to represent a \doxylink{class_rotation}{Rotation} transformation in the raytracer project. It inherits from the \doxylink{class_i_transformations}{ITransformations} interface. \begin{DoxyNote}{Note}
The \doxylink{class_rotation}{Rotation} class is responsible for applying \doxylink{class_rotation}{Rotation} transformations 
\end{DoxyNote}
